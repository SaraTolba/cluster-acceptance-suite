# Prototypes

This folder preserves older one-off scripts and test notes for reference.

These files are not used directly by the acceptance suite. They show earlier test categories, including:

- PBS example job submission and logging
- PBS output/error scanning
- module avail / module show inspection
- module dependency checks
- per-node module load checks
- SLURM MPI smoke jobs
- PBS single-node and two-node tests

Do not treat old outputs as current acceptance results.
